# BeaBoo

## Overview
BeaBoo is a static web application for story sharing and social features. The application is built as a single-page HTML file that uses external CDN services for styling and functionality.

## Project Structure
- `index.html` - Main application file containing all HTML, CSS, and JavaScript

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript (all in single file)
- **Styling**: Tailwind CSS (CDN), Font Awesome (CDN)
- **Backend Services**: Firebase (Authentication, Realtime Database, Storage)
- **Translations**: Google Translate API
- **Ads**: Google AdSense

## Running the Project
The project uses a simple Python HTTP server to serve static files on port 5000.

## Deployment
Configured for static deployment - serves files directly from the project root.
